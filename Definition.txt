Liste des définitions :
***********************


NPM : Il est installé par défaut avec Node.JS, faisant partie de son environnement. Il est le gestionnaire de paquets officiel de Node.JS et s’utilise en ligne de commande.

SASS : Sass est l’acronyme “Syntactically Awesome Stylesheets”.  C’est une extension CSS qui ajoute de nouvelles règles comme les variables, les mixins, l’héritage et plus encore, simplifiant ainsi la vie des intégrateurs web.

Grunt : Grunt a été créé par  Ben Alman et est écrit en Node.js. Il est distribué par NPM.


WebPack : C’est un taskrunner spécialisé. Il propose un système de « loader », qui permet de lier sous forme de « modules »  du code JS, CSS, HTML par exemple.

jQuery : jQuery est une bibliothèque JavaScript libre et multi-plateforme créée pour faciliter l'écriture de scripts côté client dans le code HTML des pages web.

DOM : Le Modèle Objet de Document (DOM) est une interface de programmation d'application (API) pour des documents HTML valides et XML bien-formés. Il définit la structure logique d'un document (au sens large tu terme!) et la manière d'y accéder et de le manipuler.

Compiler vs Interpreter : Un compiler traduit tout le code en binaire avant de l'éxecuter là où un interpreter va traduire le code au fur et à mesure de son éxecution.

SEO : L'optimisation pour les moteurs de recherche, couramment appelée SEO (acronyme de Search Engine Optimization), est un ensemble de techniques visant à optimiser la visibilité d'un page web dans les pages de résultats de recherche (les SERP, pour Search Engine Result Pages).

snake_case : Par opposition à CamelCase, faire du snake_case consiste à mettre un underscore "_" entre chaque mot d'une suite de mots quand on nomme quelque chose(Pour mimiquer un espace).

CamelCase : Littéralement « casse de chameau » est une pratique qui consiste à écrire un ensemble de mots en les liant sans espace ni ponctuation, et en mettant en majuscule la première lettre de chaque mot.

MarkDown : Language de balisage qui permet d'écrire du HTML de façon raccourcie. Markdown est un système d’édition et de formatage de texte ; c’est à la fois une syntaxe, un script de conversion texte → HTML et un format de fichier. Il est couramment utilisé pour les fichiers de documentation d’un projet ou d’un jeu de données -souvent nommé readme.md.
